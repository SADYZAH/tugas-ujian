# 📌 Tugas KKJ - Aplikasi Project (React)

## 📖 Deskripsi Project
Project ini adalah tugas KKJ yang dibuat menggunakan React JS.  
Aplikasi ini memiliki beberapa komponen yang saling terhubung untuk menampilkan tampilan antarmuka pengguna (UI) secara dinamis.

---

## 🧩 Fitur Aplikasi
- Menampilkan komponen UI (Header, Footer, dan Card/User)
- Menggunakan komponen React (Component-Based)
- Styling menggunakan CSS
- Struktur project sederhana dan mudah dipahami

---

## ⚙️ Penjelasan Singkat Code

### 1. App.jsx
File utama yang mengatur semua komponen.  
Di sini semua komponen seperti Header, Footer, dan UserCard dipanggil dan ditampilkan di halaman utama.

### 2. Components
Project ini menggunakan beberapa komponen:

- **Header.jsx** → Menampilkan bagian atas halaman (judul/navigasi)
- **Footer.jsx** → Menampilkan bagian bawah halaman
- **UserCard.jsx** → Menampilkan data user dalam bentuk card

---

### 3. Cara Kerja
React akan:
- Memecah UI menjadi komponen kecil
- Menggabungkan semua komponen di `App.jsx`
- Menampilkan hasilnya di browser secara realtime
